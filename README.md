# Subtext Web

Public website for [Subtext](https://chrome.google.com/webstore) — a Chrome extension for critical news reading.

## What's Here

- **[index.html](index.html)** — Landing page with product overview and support information
- **[privacy.html](privacy.html)** — Privacy policy for Chrome Web Store compliance

## Live Site

This repo is deployed via GitHub Pages:
- **Homepage:** `https://<username>.github.io/subtext-web/`
- **Privacy Policy:** `https://<username>.github.io/subtext-web/privacy.html`

## Local Development

Open the HTML files directly in your browser. No build step required.

## Deployment

Push to `main` branch. GitHub Pages automatically deploys from the root directory.

To enable GitHub Pages:
1. Go to **Settings** → **Pages**
2. Select **Deploy from a branch**
3. Choose `main` branch, `/ (root)`
4. Save

Changes go live within ~1 minute of pushing.
